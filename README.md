# Healthcare Sales Data Cleaning & Reporting Analysis

## Project Overview

This project focuses on cleaning, validating, and analyzing healthcare equipment sales data using Microsoft Excel. The dataset contains sales transactions for medical products such as Syringes, Stethoscopes, Ultrasound Devices, X-Ray Machines, Blood Test Kits, and PPE Kits across multiple regions and sales representatives.

The objective is to improve data quality, standardize reporting, and generate business insights that support management decision-making.

## Business Problem

Healthcare sales data often contains missing values, inconsistent entries, duplicate records, and formatting issues. These data quality challenges can impact reporting accuracy and business decisions.

This project addresses these issues by implementing data cleaning techniques, validation rules, calculated metrics, and reporting fields using Excel formulas and conditional formatting.

## Project Tasks

### Data Cleaning

* Identified duplicate Order IDs using Conditional Formatting.
* Highlighted missing Salesperson entries.
* Replaced missing Discount values with 0.
* Standardized Return Status values to Yes/No.
* Applied Currency (English - US) formatting to financial columns.

### Calculations

* Counted orders with Return Status = Yes.
* Calculated total orders for X-Ray Machine products.
* Computed Total Sales by region.
* Calculated Total Sales for Blood Test Kit and PPE Kit products.
* Measured Average Deductions for PPE Kit products.

### Report Enhancement

Created additional reporting fields:

* ReportName
* SalesPersonCode

These fields improve report standardization and business reporting efficiency.
